# Store: Collapsible Kit Description with Bold Title

В разделе «Готовые комплекты» интернет-магазина реализовано:

1. **Сворачиваемое описание состава комплекта** — список позиций по умолчанию скрыт под интерактивной кнопкой «Состав комплекта (N поз.)» с иконкой шеврона. При клике список разворачивается.

2. **Жирный заголовок** — строка «В комплект поставки входит...» выделяется жирным шрифтом (`font-semibold`), остальные вводные строки остаются обычными.

Логика реализована в `src/components/store/ProductDetailDrawer.tsx`:
- Используется компонент `Collapsible` из shadcn/ui
- Проверка `isTitleLine` по наличию текста «в комплект поставки входит» (case-insensitive)
- Вводный текст всегда виден над кнопкой раскрытия
